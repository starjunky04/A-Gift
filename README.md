# How it works:
scroll through the webpage to learn about this object and what it means to me
you hover over the text blocks to change the colour or reveal hidden text
you can click the pink text to hear songs

# How it was made:
Used GSAP scroll trigger to animate different CSS Classes according to how far the user scrolled.
There are scroll triggers which are hidden and when they are scrolled into frame, the animation linked to them will play.
I used CSS Hover to add styling to the text boxes and give them hover effects


Tech used: HTML CSS, Javascript, GSAP, ScrollTrigger
